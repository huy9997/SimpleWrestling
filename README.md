  Frontend:
    React.js

  Backend:
    Node.js
    Express

How to install the dependencies:

  install npm modules in client:
    1. cd into client
    2. npm install
  install npm modules for the node server:
    1. npm install

How to run the project:

  Run Frontend
  1. cd into client
  2. npm start

  Run Backend
  2. run "node index.js"
